Elements of the Program MTZTEM
==============================

Introduction
------------

The MTZTEM program library contains two programs:

		- MT3Dfwd: solves the magnetotelluric forward problem
		- ZTEM_MT3Dinv: solves the inverse problem for both magnetotelluric and/or Z-axis tipper data
